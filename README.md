# Proyecto Web: Página de Contenido del Cosmere

  

Este proyecto consiste en la creación de una página web como parte del trabajo final de la asignatura de Lenguaje de Marcas para el Grado de Técnico Superior en Administración de Sistemas Informáticos en Red (ASIR). La web está diseñada para simular una página de contenido relacionada con el universo del Cosmere, creado por el autor de fantasía Brandon Sanderson.

  

**Tecnologías Utilizadas:**

  

-  **HTML/CSS:** Para la estructura y el diseño visual de la página.

-  **JavaScript:** Para mejorar la interactividad y la funcionalidad de la web.

-  **Frameworks y Librerías:** Boostrap y JQuery.

  

## Comenzando 🚀

Para poder ejecutar el proyecto en tu maquina local se debera clonar el repositorio.

```

git clone https://github.com/DavidBuenacasa/Cosmere-web.git

```

```docker

npm start

```

  

## Despliegue 📦

  

```docker

docker build -t cosmere-web-image .

```

  

```docker

docker run -d -p 4678:3000 --name cosmere-web --restart always cosmere-web-image

```
